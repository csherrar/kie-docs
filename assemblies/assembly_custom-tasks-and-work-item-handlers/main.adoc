// Purpose statement for the assembly
As a business rules developer, you can create custom tasks and work item handlers in {CENTRAL} to execute custom code within your process flows and extend the operations available for use in {PRODUCT}.

.Prerequisites
* {CENTRAL} is deployed and is running on a web/application server.
* You are logged in to {CENTRAL}.
* Maven is installed.
* The host has access to the Internet. The build process uses the Internet for downloading Maven packages from external repositories.
